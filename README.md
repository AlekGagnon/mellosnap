# mellosnap

A new Flutter project.

## Getting Started

```bash
flutter pub get
cp .env.example .env   # then add your Supabase keys
flutter run
```

See [SUPABASE_SETUP.md](SUPABASE_SETUP.md) for auth configuration (Email, Google, Apple).

## Here is the goal of the app in french 

But : Remplacer les caméras jetables avec une application mobile. Celle-ci fonctionne en choisissant combien de photos on veut au début, après les X prochaines photos sont sauvegardées dans un fichier caché. Une fois les X photos prisent, on demande si on veut les recommencer ou on veut les recevoir par la poste. Le paiement est décidé avant et effectué après la demande de livraison avec une facture envoyé sois par mail ou juste une confirmation de paiement. Le paiement fonctionne par stripe.
